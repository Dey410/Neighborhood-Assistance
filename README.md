# 邻里帮（NeighborHelp）

**基于高德开放平台的邻里互助与本地生活服务平台**
—— 让附近的人，解决身边的事

------

## 一、项目简介

“邻里帮（NeighborHelp）”是一个基于空间位置的邻里互助平台，
面向社区、小区、校园等本地生活场景，
为用户提供 **「发布需求」** 与 **「就近接单」** 的互助服务。

项目以高德开放平台的地图与空间能力为核心，
结合 Web 技术与后端服务，
实现真实可用、可扩展的本地生活服务解决方案。

------

## 二、核心功能

### 1️⃣ 用户发布需求

- 输入需求描述与地址
- 提交邻里互助订单
- 订单自动记录地理坐标

### 2️⃣ 服务者附近接单

- 基于当前位置查询附近订单
- 按空间距离排序推荐
- 一键接单，完成服务匹配

### 3️⃣ 地图可视化展示

- 高德地图实时加载
- 订单位置 Marker 展示
- 空间位置直观可见

------

## 三、技术架构

### 前端

- Vue 3 + Vite
- Vue Router
- Axios
- 高德地图 JavaScript SDK

### 后端

- FastAPI
- SQLAlchemy
- SQLite（可扩展为 MySQL）
- RESTful API 设计

### 地图与空间能力

- 高德地图 JS SDK
- 地理坐标展示
- 距离计算与附近推荐
